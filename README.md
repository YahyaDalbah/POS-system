## prerequisites

- npm
- nodeJS
- json-server

## how to install it locally:

1. `git clone https://github.com/YahyaDalbah/POS-system.git`
2. `cd server`
3. `npm run server` (it runs the json-server with a 1s delay)
4. open a new terminal window
5. `cd client`
6. `npm install`
7. `npm run dev`
